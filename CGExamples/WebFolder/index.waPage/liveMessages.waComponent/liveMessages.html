In this example I use the open source Web Socket Server jWebsocket (http://jwebsocket.org/) to show how to send messages live to other browsers.
First download the Web Socket server from their page and unzipp it. Bevor you start the server (.../jWebSocket-1.0/bin/jWebSocketAdmin.bat) don´t forget to copy the folder "logs" next to your jWebSocket-<version> folder or the server will not start.
created by Christian Goergen with Wakanda 0.126084
If you get other problem, check out their documentation: http://jwebsocket.org/quickguide/quickguide.htm
Now you have to include the jwebsocket.js to the page. Do this in the properties in the Wakanda Studio. You can find the jWebSocket.js here: http://code.google.com/p/jwebsocket/source/browse/branches/jWebSocket-1.0/jWebSocketClient/web/res/js/jWebSocket.js
